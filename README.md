# Surya Swarup Resume Website

This package includes:
- `index.html`
- `styles.css`
- `script.js`
- `assets/images/*`
- `assets/pdfs/*`
- assignment submission write-up files

## Deploy on GitHub Pages
1. Upload all folder contents to your existing repository.
2. Commit and push the changes.
3. Open repository **Settings → Pages**.
4. Select the main branch and root folder.
5. Save and wait for the site to publish.

All links are relative, so the site will work on GitHub Pages without extra changes.

## To add the missing Market Mania deck later
1. Put the PDF inside `assets/pdfs/`.
2. Update the card in `index.html`.
3. Add a thumbnail in `assets/images/` if you want a preview cover.
